AMR Multiphase is a high-performance C++ solver for incompressible two-phase fluid flows. It uses Adaptive Mesh Refinement (AMR) to efficiently resolve sharp gradients in the solution â€” automatically refining grids where needed and coarsening them elsewhere. The solver supports both Newtonian and non-Newtonian (viscoelastic Oldroyd-B) fluid models, and is second-order accurate in $L^1$, $L^2$, and $L^\infty$ norms.

## What this library solves

AMR Multiphase implements a staggered-grid (MAC) discretization of the incompressible two-fluid mixture equations. Each fluid phase â€” a **network** phase and a **solvent** phase â€” carries its own velocity field and volume fraction. The two phases interact through a drag coupling term.

## Getting started

<Steps>
  <Step title="Build IBAMR and dependencies">
    Install [IBAMR](https://ibamr.github.io/building) and its dependencies. You can use [autoibamr](https://github.com/ibamr/autoibamr) for an automated build.
  </Step>
  <Step title="Compile the library">
    Configure with CMake pointing to your IBAMR installation, then build the `MultiphaseLib` target and any example executables.
  </Step>
  <Step title="Configure your simulation">
    Create an input file specifying grid geometry, fluid parameters, time stepping settings, and solver configuration.
  </Step>
  <Step title="Run and visualize">
    Execute the compiled binary with your input file. Visualize results with [VisIt](https://visit-dav.github.io/visit-website/).
  </Step>
</Steps>

## Key features

- **Second-order accuracy** â€” verified in $L^1$, $L^2$, and $L^\infty$ norms for all solution variables
- **Adaptive Mesh Refinement** â€” gradient-based tagging automatically refines near sharp interfaces
- **Multigrid preconditioner** â€” efficient FAC preconditioner solves the coupled saddle-point system
- **Viscoelastic support** â€” Oldroyd-B conformation tensor model via the `CFMultiphaseOldroydB` strategy
- **Multiple limiters** â€” UPWIND, CUI, FBICS, and MGAMMA convective operator limiters
- **Parallel-ready** â€” built on IBAMR/SAMRAI/PETSc for MPI-parallel computation
